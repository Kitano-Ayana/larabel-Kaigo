

## 介護士の登録や患者様の状態を把握できるアプリ「Care-Wing」
　　介護の現場において何よりも大切なことは患者様の状態を把握し共有することです。毎日の状態を記録することで体調の変化に気づくことができ迅速に対応することができます。そのためにはより簡単に記録し、日々の変化が見やすいサービスが必要だと思い、このサイトを作成しました。
